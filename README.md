## Setup

1. Prerequisites

    ```bash
    $ sudo apt-get install python3-pip
    $ pip install psutil
    ```

1. Test High CPU Load & kill the process using python script

    ```bash
    $ cd python-automation/high_cpu_load
    $ bash load.sh &
    $ python3 kill_process.py
    ```
